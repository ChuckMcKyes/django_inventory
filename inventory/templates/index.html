Three web apis are served. These can be copied into a browser address bar to serve up data in json format, or can be used in other apps:

https://yosa.design/inventory/api/list/

will serve the entire list of inventory items.

https://yosa.design/inventory/api/list/?search=ca

will search for those items with "ca" in their names.

https://yosa.design/inventory/api/categories/

will show the list of item categories

https://yosa.design/inventory/api/item/2

will show the details for the item with id=2, for example.
